## POST 二课活动报名及取消

POST //act/actInfo/v1.0.0/updateActMemberStatus

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|actId|query|integer| 否 |活动id|
|memberId|query|integer| 否 |成员id|
|status|query|integer| 否 |状态，1为报名，2为取消报名|
|validCodd|query|integer| 否 |验证码|
|x-access-token|header|string| 否 |none|
