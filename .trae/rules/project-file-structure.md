#### 项目文件结构参考：
根据业务，本app一共分为四大独立模块和一个公共模块
四大独立模块分别是：
- 首页
- 课表
- 广场
- 我的
具体体现到项目的文件结构中为：
```
nongyu/
├── src/
│   ├── modules/
│   │   ├── common/
│   │   ├── Home/
│   │   ├── Schedule/
│   │   ├── Square/
│   │   ├── My/
```
其中，在每个子模块下，主要包含以下几个文件：
- `index.tsx`：模块的入口文件，用于导出模块的主要组件
- `pages/`：模块的页面组件目录，包含该模块下的所有页面组件
- `components/`：模块的组件目录，包含该模块下的所有组件，非整个项目的全局组件，一般仅在该模块下使用
- `utils/`：模块的工具函数目录，包含该模块下的所有工具函数（可选）
- `constants/`：模块的常量目录，包含该模块下的所有常量（可选）
- `styles/`：模块的样式目录，包含该模块下的所有样式（可选）
- `hooks/`：模块的自定义钩子目录，包含该模块下的所有自定义钩子（可选）
