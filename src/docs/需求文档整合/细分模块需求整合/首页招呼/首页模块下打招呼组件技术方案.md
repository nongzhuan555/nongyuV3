# 首页打招呼组件技术方案（简化版）

## 文档信息
- 版本：v1.1-简化
- 关联PRD：同目录《首页打招呼组件（简化版 PRD）》
- 目标：以最小实现落地“天气 + 时段 + LLM 生成问候语”，选型明确、接口简洁、便于后端集中接入

## 选型结论
- 天气服务（前端直连，推荐）：Open-Meteo，免费无钥匙，全球覆盖。接口返回 WMO weathercode 与温度，前端在 `shared/weather.ts` 已实现中文映射与缓存。示例：`GET https://api.open-meteo.com/v1/forecast?latitude=30.67&longitude=104.07&current_weather=true&timezone=Asia%2FShanghai`。官网：https://open-meteo.com/
- 大模型服务（后端对接，推荐）：DeepSeek 官方 API，价格低、中文效果好，OpenAI 兼容样式，适合自建后端转发；官方定价与模型信息见 Models & Pricing（https://api-docs.deepseek.com/quick_start/pricing）。亦可通过腾讯云 DeepSeek 接入（https://cloud.tencent.com/document/product/1772/116385）。

## 极简调用链
- 前端直连 Open-Meteo 获取“天气现象与温度”，并本地中文映射与30分钟缓存。
- 前端本地计算“时间段”。
- 前端将“昵称/校区/时段/天气摘要/温度”发给后端，由后端调用大模型生成≤30字单句（失败则模板）。

## 客户端最小实现
- 时间与时段：`src/shared/time.ts`
- 已提供方法：
  - `getDeviceTime(): Date` 返回设备当前时间
  - `getISONow(): string` 返回当前 ISO 字符串
  - `getTimeSlotLabel(d?: Date)` 返回 `凌晨/清晨/上午/中午/午后/傍晚/夜间`
- 使用示例（伪码）：
```ts
import { getDeviceTime, getTimeSlotLabel } from '@/shared/time';
const now = getDeviceTime();
const timeSlot = getTimeSlotLabel(now);
```

- 天气获取：`src/shared/weather.ts`
- 已提供方法：
  - `getWeatherByCampus(campus: string): Promise<{ summary: string; temp?: string }>`
- 行为：基于校区名称映射经纬度 → 调用 Open-Meteo current_weather → WMO 代码映射中文摘要 → AsyncStorage 缓存 30 分钟


## 后端接口契约（最小版）
- 路由：POST `/api/greeting`
- 请求体：
```json
{
  "name": "张三",
  "campus": "雅安校区",
  "timeSlot": "午后",
  "weather": "小雨",
  "temp": "12℃"
}
```
- 后端行为：
  - 使用传入的天气与时段构造提示词，调用 DeepSeek 生成≤30字中文单句（建议超时2–3s）；失败或超时走模板。
- 响应体：
```json
{
  "content": "张三你好，现在是午后，雅安校区小雨，出门记得带伞",
  "createdAt": "2026-02-15T06:00:00.000Z"
}
```

## 天气与大模型最小示例（后端参考）
- DeepSeek 官方 API（OpenAI 兼容）：
  - `POST https://api.deepseek.com/chat/completions`
  - 形如：
```json
{
  "model": "deepseek-chat",
  "messages": [
    { "role": "system", "content": "用简体中文输出≤30字单句问候语，不含emoji，标点≤2" },
    { "role": "user", "content": "姓名: 张三；校区: 雅安校区；时段: 午后；天气: 小雨, 12℃" }
  ],
  "temperature": 0.8
}
```
  - 定价与模型能力以官方页面为准（https://api-docs.deepseek.com/quick_start/pricing），或参考腾讯云 DeepSeek 文档（https://cloud.tencent.com/document/product/1772/116385）。

## 降级与约束
- 大模型失败/超时：模板文案 `{name}你好，现在是{timeSlot}，{campus}{weather}{tempHint}，祝你顺利。`；若无天气则省略天气片段。
- 天气失败：不阻塞生成，`weather=''` 继续走大模型；大模型再失败则模板。
- 客户端仅做展示，不存储密钥与模型返回日志。

## 接入步骤（极简）
1. 前端完成 `shared/weather.ts`（已实现）与 `shared/time.ts` 接入，在首页调用获取 `weather/temp/timeSlot`。
2. 后端实现 `POST /api/greeting`，调用 DeepSeek 生成文案（或模板降级）。
3. 首页集成：将昵称/校区/时段/天气/温度传后端 → 展示返回文案；无网/失败时使用模板。
